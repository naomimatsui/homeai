# HomeAI

家の持ち物を「登録して探せる」アプリ。単一ファイル（`index.html`）、データは各端末の **localStorage** に保存（サーバー無し／個人ごとに完全分離）。

- 公開URL：https://naomimatsui.github.io/homeai/
- リポジトリ：naomimatsui/homeai（GitHub Pages）

## 主な機能
- 登録（品名＋場所）・自動採番・全文検索
- 部屋登録＋2階層グループ表示（部屋 → その中の場所）
- 写真・音声入力・音声検索（読み上げ）
- 一括インポート（Excel/CSV/テキスト）
- 在庫アラート（品目ごとの上限＝買いすぎ／下限＝買い足し）・在庫±
- 季節タグ・探した時間カウンター・30日チャレンジ・30日プラン
- 🔖 **バーコード**（登録／さがす両対応・任意）＋メモ
- ⭐ **大切なものだけ登録**（失くしたら困るものだけ管理＝きれいに暮らす人・防災/相続/保険にも）

## 大切なものだけ登録（全部登録しなくていい）
> 「全部じゃなくていい。大切なものだけ、HomeAIに覚えさせよう。」

片付けが苦手な人だけでなく、**収納が整っている人**でも使えます。通帳・印鑑・保険証券・パスポート・年金手帳・契約書・保証書・薬・防災用品・予備の鍵・思い出の品・高価なもの…**失くしたら困るものだけ**登録すれば、防災・相続・保険・緊急時に役立ちます。

- 登録タブ上部の「⭐ 大切なものを登録する」から始める（大切フラグが自動ON）
- または登録フォームの「⭐ 大切なもの」チェックを付けて保存
- さがすタブの「⭐ 大切なものだけ表示」で、大切なものだけ一覧できる
- 一覧では ⭐大切 バッジが付く

## バーコード機能の動作確認手順

### A. 登録タブ
1. **スキャンで入力**：「🔖」ボタン → カメラでJANコードを読み取る → バーコード欄に番号が入る
   - ※カメラが使えない環境でも **バーコード欄に番号を手入力** できます（必須ではありません）
2. 商品名・場所・個数・在庫警告数（上限/下限）・メモ を入力
3. 「＋ 登録する」で保存
4. **2回目以降**：同じバーコードを読む/入力すると、前回の **商品名・場所が自動で候補入力** される（個数だけ変えて保存しやすい）

### B. さがすタブ
1. **バーコードでさがす**：「🔖 バーコードで探す」→ 商品を読み取る
   - 家にある → 「商品名：場所／残り○個（在庫少なめ⚠️）」を表示、一覧に該当品を表示
   - 家に無い → 「家には登録されていません／この商品を登録しますか？」＋**［登録タブで登録する］ボタン**（押すとバーコードを引き継いで登録タブへ）
2. **キーワード検索**：検索窓に品名・場所・メモ・**バーコード番号**を打っても検索できます

### エラー安全性
- カメラ・ライブラリが使えなくても、手入力での登録／検索は動作します。
- バーコード処理でエラーが出てもアプリ全体は停止しません（ライブラリ未読込チェック＋例外catch）。

## データ項目（localStorage）
キー：`homeai_items_v1`（配列）。1件の項目：

| 仕様の名称 | 本アプリの内部キー | 内容 |
|---|---|---|
| id | `id` | 一意ID |
| name | `name` | 品名（自動採番あり：ワンピース1 等） |
| location | `place` | 場所（「リビング / クローゼット1」等） |
| quantity | `qty` | 個数 |
| alertQuantity | `hi`（買いすぎ上限）/ `lo`（買い足し下限） | 在庫警告数 |
| barcode | `barcode` | バーコード番号（任意） |
| photo | `photo` | 写真（dataURL・任意） |
| memo | `memo` | メモ（任意） |
| （大切フラグ） | `important` | ⭐大切なもの true/false（任意） |
| createdAt | `created` | 作成時刻 |
| updatedAt | `updatedAt` | 更新時刻 |

> ※ 既存アプリが `place/qty/hi` で稼働中かつ既存データがあるため、内部キーは維持（リネームによる全体停止・データ消失を回避）。上表が仕様名との対応です。

補助データ：
- `homeai_barcode_dict_v1`：`{ バーコード番号: { name, place } }`（再スキャン時の自動入力用）
- `homeai_rooms_v1` / `homeai_stats_v1` / `homeai_challenge_v1` / `homeai_plan_v1`
